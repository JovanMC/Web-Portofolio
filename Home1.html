Car Gallery Website


Car Gallery Website merupakan website statis yang dibangun dengan menggunakan HTML dan CSS (Bootstrap4).
Pada website ini menyajikan Gambar Mobil yang akan memberikan informasi terkait Mobil yang tersedia pada sebuah showroom mobil.

SKApp


SKApp adalah aplikasi Kasir (POS) untuk Android yang dibangun dengan bahasa pemrograman Java dan menggunakan Android Studio.

Aplikasi ini mempunyai beragam fitur antara lain :

- Penyimpanan Data Barang (CRUD)

- Membuat dan menyimpan data Transaksi Barang

- Mengelola Laporan Bulanan

- Mengelola Utang dan Piutang

- Menyimpan dan mengelola data pegawai

KampusKu


KampusKu merupakan aplikasi CRUD (Create, read, update dan delete) sederhana menggunakan framework CodeIgniter 4.
Aplikasi ini memiliki fitur untuk mengelola data mahasiswa.
